# Online Retail - Product Performance Dashboard

An interactive **Product Analysis Dashboard** built in Microsoft Excel using the UCI Online Retail dataset.

## Project Overview

This dashboard provides a clear and dynamic view of sales performance for a UK-based online retailer. It allows users to analyze revenue trends, top-performing products, and geographic sales distribution through interactive visualizations.

## Dataset
-**Source**: UCI Machine Learning Repository - Online Retail Dataset

- `Dashboard.xlsx` – Main interactive Excel file
- `Dashboard_Screenshot.png` – Final dashboard image
- `README.md` – This file

## How to Use

1. Download and open `Online_Retail_Dashboard.xlsx`
2. Use the **Timeline** on the right side
3. All KPIs and charts will update automatically
